限量《漫威星際異攻隊》Epic Game 遊戲虛寶 回饋活動

限量《漫威星際異攻隊》Epic Game 遊戲虛寶 回饋活動

GeForce NOW 會員獎勵！獨家限量回饋，GeForce NOW 的用戶們可領取《漫威星際異攻隊》「時尚星爵服裝」(Sleek Lord Outfit) 遊戲虛寶，數量有限、送完為止，僅適用於Epic Games 數位商店且需已購買該款遊戲。

只要你是台灣大哥大 GeForce NOW 的基本方案或是白金方案用戶，都可以領取活動獎項，此獎項於 Epic Games 數位商店兌換的截止日期為 2022/01/06。

請注意！您的Epic Game遊戲庫內需已擁有《漫威星際異攻隊》遊戲（遊戲需自行購買），即可馬上兌換遊戲內酷炫限量的「時尚星爵服裝」！
*須遵守活動條例與細則

活動期間
‧ 2021/11/23 (12:00pm)～2022/01/06 (23:59pm)

活動辦法
‧ 欲領取「時尚星爵服裝」(Sleek Lord Outfit) 獎項，請依以下步驟

  1. 註冊並登入 你的GeForce NOW會員帳號
  2. 至 會員方案 訂閱基本或白金方案
  3. 點選下方「點我領取遊戲虛寶」
  4. Copy 複製該遊戲虛寶序號
  5. 至 Epic Games 數位商店 ，登入你的 Epic Games 帳號
  6. 至 兌換代碼 頁面，兌換你的遊戲虛寶序號
  7. 輸入你的遊戲虛寶序號
  8. 透過 GeForce NOW 打開遊戲，時尚星爵服裝將兌換至你的遊戲中
