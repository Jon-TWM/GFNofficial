《原神》獨家會員限量好禮又來囉 🎁
透過 GeForce NOW 雲端串流引領你一起探索廣大的「提瓦特」～
立即領取《原神》遊戲虛寶禮包，數量有限、送完為止！

此獎項兌換期限至 5/18 23:59pm 為止 ，請於領取後盡快兌換！

  • 大英雄的經驗 (Hero's Wit) x 3
  • 30,000 摩拉 (Mora)
  • 精鍛用魔礦 (Mystic Enhancement Ore) x 5

    * 序號兌換截止期限至 5/18 23：59pm 為止

活動方法
欲領取《原神》虛寶禮包，請依以下步驟：
  1. 點擊下方「點我領取遊戲虛寶」
  2. Copy 複製該遊戲虛寶序號
  3. 前往 Hoyoverse兌換碼頁面 輸入兌換碼
  4. 兌換成功後獎項內容便會出現在你的遊戲中，祝您遊玩愉快！

常見活動問題
問：我是因為其他相關 GeForce NOW 的優惠活動所申辦的會員到這裡來，請問我也可以參加此獎勵活動？
答：可以。但僅限獲獎一次。
問：我該如何符合 GeForce NOW 獎勵的領取資格？
答：你只要擁有台灣大哥大 GeForce NOW 會員訂閱，不論基本方案或白金方案皆可領取限量獎勵。僅限獲獎一次。
問：如果我符合所有資格和要求，是否能保證獲得獎勵？
答：活動獎勵採先搶先贏制，數量有限，無法保證所有人都能得到。
問：如何知道什麼時候有新的獎勵？
答：請隨時關注並追蹤我們官方網站或 臉書粉絲團 ，以便隨時掌握活動的最新消息。
問：我可以把我的《原神》遊戲虛寶禮包給其他玩家嗎？
答：如果您已經至 遊戲內兌換遊戲虛寶禮包，就無法做轉移到其他玩家了。
問：我螢幕顯示有錯誤訊息，我該怎麼辦？
答：請重新再嘗試一次，如果還是無法解決問題的話，請麻煩至官網”我要發問”與我們聯繫。
