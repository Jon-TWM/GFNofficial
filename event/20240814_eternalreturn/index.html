《永恆輪迴》抽獎活動
活動期間
  • 2024/9/3 (12:00pm)～2024/9/30 (23:59pm)
活動辦法
    活動1
    人人有獎：只要於活動期間首次登入《永恆輪迴》，即可獲得 GeForce NOW 希爾維婭表情包一組。
    * 不僅限使用 GeForce NOW，使用任何設備登入皆可獲得
    * 獎項將由遊戲系統自動派發於遊戲內信箱

    活動2
    ：符合下述步驟資格與完成指定任務，即可至 我要參加 填寫資料，將由系統隨機抽出得獎者，有機會獲得《永恆輪迴》遊戲虛寶包序號與 MSI 筆電。

  • 活動單位確認得獎名單時，得獎者須保有各資格的 GeForce NOW 會員訂閱身份。

活動步驟

獎項
  • 《永恆輪迴》虛寶包：造型 DATA 箱 x 鑰匙 (5 組) + 5000 A-Coin (市價價值 NTD $450元) (限量 150 份)
  • MSI筆電 (市價價值 NTD $?????元) (乙台)
步驟一
    資格1：所有 GeForce NOW 會員 ( 立即申辦 )
    於活動期間，會員透過 GeForce NOW 登入《永恆輪迴》遊戲一次，即可登記抽虛寶包一組。
    * 後台需成功有遊玩《永恆輪迴》的紀錄
    * 序號兌換方法：遊戲大廳主頁右上方「選項」→「服務」→「輸入兌換券」中輸入兌換序號
    * 虛寶兌換期限至：2024/12/31止

    資格2：鈦金方案與白金方案會員* ( 立即申辦 )
    於活動期間，鈦金方案或白金方案會員透過 GeForce NOW 登入《永恆輪迴》遊戲一次，登記後即可再自動加碼抽MSI筆電乙台。
    * 不包含日訂會員
    * 後台需成功有遊玩《永恆輪迴》的紀錄
    * 可與資格1重複獲獎
步驟二
    至 我要參加 登記資料，即可參加抽獎活動
獎項發贈
  • 將於 2024/10/15 前，公布得獎名單於下方 得獎名單處。
  • 於公布後翌日起，五個工作天內，透過得獎者於此活動網頁所報名的電子信箱通知得獎訊息。
    * 為了確保能與得獎者聯繫，以及將獎項寄送出，請確保您所留的聯絡資訊正確。
    * 虛寶包序號得獎者：獎品序號將同得獎通知一起透過得獎者於活動網頁所登記的電子信箱發送。
    * 筆電得獎者：若於2024/11/15 前未收到得獎者的回信或應收到的資料，逾期將視為棄權得獎資格，並不另行再次抽獎。
    * 若因個人因素無法確實接收電子郵件得獎通知，恕不補發。
    * 活動單位確認得獎名單時，得獎者須保有個別活動資格之方案訂閱身分。
